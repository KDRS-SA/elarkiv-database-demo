KDRS/KAI
- arkivplan.no skjema for elektronisk system
- InfoOmInnlevering.doc fra "prosesser og rutiner for digitalt depot"

DIAS:
- INFO      : Overordnet informasjon ved innlevering (deponering/avlevering).
- ADDML     : Teknisk beskrivelse av poststrukturerte datafiler (tabelluttrekk).
- METS      : Definerer arkivpakkes indre struktur og ytre beholder ("container").
- PREMIS    : Bevarings-metadata og annen forståelsesinformasjon.
- EAD       : Beskriver bevarings-objekter logisk som arkivmateriale.
- EAC-CPF   : Beskriver arkivskapere og andre aktører knyttet til arkivokbjekter.
